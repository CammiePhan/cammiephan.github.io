细品咖啡

notes
  |  
 阅读次数

有一种咖啡值得你细细慢慢地品尝。是什么咖啡呢？且等我端起一杯浓香的咖啡，听我细细道来。

哪种咖啡？

其实，南美州有一种美味的咖啡叫做 java。我这里说的咖啡便是 java啦！

1991年，SUN公司为进行嵌入式开发，基于C++ 开发出一种新平台，称为 Oak，即为 JAVA的前身。之后，SUN公司由 Oak开发出了一种 Hotjava浏览器。基于Oak又产生了一种新的语言，即现在的 java。为它命名的时候，命名者想起了一种经常喝的南美州咖啡 java。于是，java语言正式诞生了。

让我们遇见JDK

JDK（全称是 Java Development Kit）是 java语言的软件开发工具包，主要用于移动设备、嵌入式设备上的 java应用程序。运行环境包括Windows, Linux, IOS, Mac OS.

JDK的版本包括

  • SE(J2SE), standard edition, 标准版，从JDK 5.0开始改名为Java SE
  • EE(J2EE), enterprise edition, 企业版，从JDK 5.0 开始改名为Java EE
  • ME(J2ME), micro edition, 主要用于移动设备和嵌入式开发设备上的 java应用程序，从 JDK5.0开始改名为Java ME

JDK包含的组件有

  • javac，编译器，将源程序转为字节码
  • jar，打包工具，将相关的类文件打包成一个文件
  • javadoc，文档生成器，从源代码注释中提取文档
  • 等等等等

让我们遇见JRE

JRE（全称是 Java Runtime Environment）指java运行环境，运行JAVA程序所必须的环境集合，包括java虚拟机、JAVA核心类库和支持文件。
